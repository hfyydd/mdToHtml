# 微信公众号文章示例

这是一个示例公众号文章，用于测试 MD2HTML 工具的批量转换功能。

## 为什么使用 Markdown

Markdown 是一种轻量级标记语言，创建于2004年。它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML（或者HTML）文档。

## Markdown 的优势

1. **简洁明了**：语法简单，容易上手
2. **格式丰富**：支持标题、列表、代码块等丰富的格式
3. **易于转换**：可以轻松转换成HTML、PDF等格式
4. **跨平台**：在任何文本编辑器中都可以编写
